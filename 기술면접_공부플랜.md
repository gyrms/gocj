## 1순위 - JPA 심화

**추천 교재**: 김영한 - 자바 ORM 표준 JPA 프로그래밍 (인프런)

### 체크리스트

#### 영속성 컨텍스트
- [ ] 영속성 컨텍스트란 무엇인가
- [ ] 1차 캐시 동작 원리
- [ ] 쓰기 지연 (Write-Behind) 동작 원리
- [ ] 변경 감지 (Dirty Checking) 동작 원리
- [ ] 플러시(Flush) 시점과 동작

#### N+1 문제
- [ ] N+1 문제가 왜 발생하는지 설명할 수 있다
- [ ] fetch join으로 해결하는 방법
- [ ] @EntityGraph로 해결하는 방법
- [ ] @BatchSize로 해결하는 방법
- [ ] 세 가지 방법의 차이와 각각 언제 쓰는지

#### 로딩 전략
- [ ] 지연 로딩(LAZY) vs 즉시 로딩(EAGER) 차이
- [ ] 기본값이 무엇인지 (@OneToMany, @ManyToOne 각각)
- [ ] 실무에서 왜 LAZY를 권장하는지

#### 연관관계 매핑
- [ ] @OneToMany, @ManyToOne, @OneToOne 매핑 방법
- [ ] 연관관계 주인(Owner) 개념
- [ ] 양방향 매핑 시 주의사항
- [ ] cascade 옵션 종류와 사용 시 주의점

#### 트랜잭션
- [ ] @Transactional 동작 원리 (프록시 기반)
- [ ] 전파 옵션 (REQUIRED, REQUIRES_NEW 등)
- [ ] readOnly = true 의 의미와 성능 이점
- [ ] 자기 호출(Self-invocation) 문제

#### QueryDSL
- [ ] QueryDSL 기본 사용법
- [ ] 동적 쿼리 작성 (BooleanBuilder, BooleanExpression)
- [ ] JPQL vs QueryDSL 차이
